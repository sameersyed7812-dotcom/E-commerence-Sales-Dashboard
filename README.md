# E-commerence-Sales-Dashboard

This project is an interactive E-Commerce Sales Dashboard built using Power BI. It provides insights into sales performance, profit trends, customer behavior, and payment patterns to support data-driven decision-making.

📌 Key Features
📈 Sales & Profit Analysis
🏆 Top Performing States & Customers
🛍️ Category & Sub-Category Insights
💳 Payment Mode Distribution
📅 Monthly Profit Trends
🎛️ Interactive Filters (Quarter & State Selection)
📊 Dashboard Insights
Total Sales: 438K

Total Profit: 37K

🔍 Key Observations:
Maharashtra generates the highest revenue
Clothing category contributes the most sales (~62%)
COD is the most used payment method (~43%)
Profit varies significantly across months (seasonal trend)
Printers, Sarees, and Bookcases are top sub-categories
